# 紅陽科技 WooCommerce 金流模組
## WooCommerce v3.x / v4.x / v5.x 模組
※請進入「v3.x_4.x」資料夾下載
#### 支援金流：
1. 信用卡刷卡（含分期付款）
2. Apple Pay
3. Google Pay
4. 銀聯卡刷卡
5. 超商繳費單繳費
6. 超商代碼繳費
7. 超商取貨付款
8. ATM轉帳（虛擬帳號）
9. 網路ATM轉帳
10. 台灣Pay

#### 支援物流：
1. 7-11超商
2. 全家超商
3. 萊爾富超商
4. OK超商
5. 宅配通


## WooCommerce v2.6.5 / WordPress v4.7.5
#### 本版本不再更新，建議您升級到新版本以取得完整的金流物流服務支援
※請進入「v2.6」資料夾下載
#### 支援紅陽支付
1. 信用卡付款（含分期付款，可超商取貨）
2. 銀聯卡付款
3. 網路ATM付款（可超商取貨）
4. 超商繳費單付款（可超商取貨）
5. 超商代碼付款（可超商取貨）
6. 超商取貨付款（7-11超商）
